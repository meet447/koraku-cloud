## 2026-06-26 - Adding Tooltips to Icon-Only Buttons
**Learning:** Icon-only buttons with `aria-label`s for screen readers are still inaccessible to sighted mouse users if they lack a visual tooltip. Responsive hiding classes (e.g., `md:hidden`) can visually collapse buttons to just their icons on small screens while still keeping them active on larger screens.
**Action:** Always provide a `title` attribute containing the same descriptive text as the `aria-label` for any button that functions purely as an icon visually, ensuring accessibility across different assistive technologies and user preferences.
